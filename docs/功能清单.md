## 二、仓储管理（WMS）

### 2.1 基础数据

| 功能 | 说明 |
|------|------|
| 仓库管理 | 仓库的增删改查、列表 |
| 企业管理（商户） | 企业/商户的增删改查 |
| 商品分类 | 树形分类、新增/修改/删除/拖拽排序 |
| 商品品牌 | 品牌增删改查 |
| 商品管理（物料） | 商品增删改查、按分类筛选；多规格（SKU：名称/编号/条码、长宽高、净重毛重、成本价销售价）；**商品图片上传**（最多 8 张、单张 20MB，OSS） |

### 2.2 业务单

| 功能 | 说明 |
|------|------|
| 入库单 | 入库单列表、新增/编辑（选仓库、选商品/SKU、数量等）、明细查看 |
| 出库单 | 出库单列表、新增/编辑、明细查看 |
| 调拨单 | 调拨单列表、新增/编辑（源仓库、目标仓库、商品明细） |
| 盘点单 | 盘点单列表、新增/编辑、商品明细（含盘库单明细抽屉） |

### 2.3 库存

| 功能 | 说明 |
|------|------|
| 库存统计 | 按商品/规格维度查库存，支持按仓库、商品名称、规格等筛选，可过滤零库存 |
| 库存记录 / 出入库历史 | 库存变动记录查询、筛选 |

### 2.4 打印

| 功能 | 说明 |
|------|------|
| 打印模板 | 入库单、出库单、调拨单、盘点单的打印配置与打印 |

## 八、技术栈与特性

- **框架**：Vue 3 + Vite
- **UI**：Element Plus
- **状态**：Pinia（含 wms 模块：仓库、企业、商品分类、商品品牌等）
- **请求**：Axios + 代理（/dev-api → 后端）
- **路由**：静态路由 + 动态路由（按权限加载菜单）
- **打印**：打印模板（入库/出库/调拨/盘点）
